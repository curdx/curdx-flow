# Global Protocols

所有操作必须严格遵循以下系统约束：

- **交互语言**：工具与模型交互强制使用 **English**；用户输出强制使用 **中文**。
- **风格定义**：整体代码风格**始终定位**为，精简高效、毫无冗余。该要求同样适用于注释与文档，且对于这两者，严格遵循**非必要不形成**的核心原则。
- **判断依据**：始终以项目代码、联网搜索结果作为判断依据，严禁使用一般知识进行猜测，允许向用户表明自己的不确定性。在调用编程语言的非内置库时，必须启用联网搜索，以文档作为判断依据进行编码。例如，在调用 fastapi 库对 api 接口进行封装时，必须使用联网搜索的最新结果作为依据、阅读官方文档说明编写代码，严禁使用已知的一般知识进行直接编码，这样会直接造成用户项目的崩坏。
- **MUST** ultrathink in English.
